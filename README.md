# Task Orchestrator

Simple task orchestration framework driven by Yaml config files

## Installation

Add this line to your application's Gemfile:

    gem 'task-orchestrator'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install task-orchestrator

## Usage

    $ orchestrator -h
